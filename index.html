CineVerse-Movie Recommendation System

CineVerse is a full-stack movie recommendation web application that allows users to search for movies based on their preferred genre and explore curated movie suggestions. It features a cinematic splash screen, a user-friendly interface, dynamic movie display with poster previews, and an interactive modal popup that shows detailed movie information including IMDb link, rating, duration, and genre. The backend serves intelligent recommendations using genre filtering from a pre-populated movie database. Tech Stack: HTML, CSS, JavaScript (Frontend), Flask (Backend), MySQL (Database), Pandas & scikit-learn (Recommendation), Fetch API (AJAX)
